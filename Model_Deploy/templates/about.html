About Application

This application is intended to be used by Dermatology specialists in delivering skin cancer diagnosis.
This application is capable of detecting seven types of skin cancers,namely:

1. Melanoma

2. Melanocytic Nevus

3. Basal cell carcinoma

4. Actinic Keratosis/ Bowens' disease

5. Benign Keratosis

6. Dermatofibroma

7. Vasuclar Lesion

The AI model used in diagnosing the skin cancer type has an overall accuracy of 80.45%. The precision with which model predicts for each of the classes varies.
